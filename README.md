# 🎵 Music Speed Studio

Página web para modificar audio directamente en el navegador — sin servidores, sin instalaciones.

## ¿Qué hace?

- Sube MP3, MP4, WAV, OGG o AAC
- Ajusta **velocidad** (0.25× – 4×)
- Ajusta **pitch** independiente (±12 semitonos)
- **Bass Boost** con filtro low-shelf
- **Normalización** de volumen
- Visualizador de frecuencias en tiempo real
- Descarga el resultado en formato WAV

## Tecnología

HTML + CSS + JavaScript puro — usa la **Web Audio API** nativa del navegador.  
No requiere dependencias ni build step.

## Deploy

Desplegado en **Cloudflare Pages** — live en: `https://<tu-proyecto>.pages.dev`

## Uso local

Solo abre `index.html` en cualquier navegador moderno.
